# 自制扫描隧道显微镜

## 工作顺序

1. 翻译其他作者写的STM文档 OK
2. 将元器件替换为国产
3. 基于其他作者的PCB重新设计和打印PCB
4. 调试电子设备
5. 基于其他作者的结构重新设计和制作隔振结构
6. 重写软件(Rewrite In Rust!)
7. 系统整体调试
8. 扫描图像
